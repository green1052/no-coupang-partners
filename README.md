# no-coupang-partners

쿠팡 파트너스와 관련된 것을 차단합니다.

## 기능

물건을 검색할 때마다 나오는 쿠팡 파트너스 사이트를 숨깁니다.

쿠팡 파트너스 추적 링크를 차단합니다.

## 설치

해당 주소를 복사해 AdGuard, uBlock Origin으로 설치

```
https://raw.githubusercontent.com/green1052/no-coupang-partners/master/filter.txt
```
